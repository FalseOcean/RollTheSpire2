# WPF UI Current Guide

The current WPF interface contains:

- Single seed analysis
- Batch seed search
- Search results
- Event encyclopedia
- Data status
- Favorites database
- Candidate pool

The batch search UI is organized around:

- Basic run settings
- Seed source
- Neow / process-oriented filters
- Final-result filters
- Event filters
- Boss / Ancient filters
- Search controls
- Result actions

- Ironclad = 铁甲战士
- Defect = 故障机器人
- Silent = 静默猎手
