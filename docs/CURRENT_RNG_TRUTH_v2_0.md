# Current RNG Truth for STS2 v0.107.1+

This document records the current verified RNG assumptions used by RollCore.

## Seed

- Visible seed remains 10 characters by default.
- `I -> 1`, `O -> 0` normalization remains relevant.
- Seed string is converted with deterministic hash and cast to uint.

## RNG implementation

- STS2 v0.107.1+ uses MegaRandom / xoshiro256** with SplitMix64 initialization.
- RollCore mirrors this behavior for external prediction.

## Run RNG streams

Run streams are derived from:

```text
uint(hash(seed)) + uint(hash(snake_case_stream_name))
```

Examples:

- `up_front`
- `act_selection`
- `niche`

## Player RNG

v0.107.1+ PlayerRng uses player slot index, not NetId:

```text
player_seed = hash(seed) + player_slot_index
```

Singleplayer uses slot index 0.

## Event RNG

```text
event_seed = uint((int)hash(seed) + (isShared ? 0 : player_slot_index)) + uint(hash(event_id))
```

Neow is an `AncientEventModel`, `IsShared=false`, event id `NEOW`.

## Act selection

Act selection uses independent stream:

```text
new Rng(hash(seed), "act_selection")
```

Act buckets:

- Act1: `Overgrowth`, `Underdocks`
- Act2: `Hive`
- Act3: `Glory`

## Boss and Ancient order

During Act room generation:

```text
event queue shuffle
weak encounter generation
regular encounter generation
elite encounter generation
boss = NextItem(AllBossEncounters)
ancient = NextItem(localAncients + sharedAncientSubset)
A10 Act3 second boss after ancient
```

Boss pool comes from the boss encounters in `GenerateAllEncounters`, not `BossDiscoveryOrder`.

## Event effective queue

Raw event queue is generated by shuffling `act.AllEvents + shared events` after epoch filtering.

Actual ordinary event prediction uses the effective queue:

```text
eventsVisited offset
EnsureNextEventIsValid
Hook.ModifyNextEvent
AddVisitedEvent(final)
MarkRoomVisited(Event)
```

Each Act's starting Ancient uses `EventRoom`, so ordinary events start from `raw[1]` after the starting Ancient is entered.
